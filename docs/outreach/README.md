# Partner Outreach Templates

3 partnership emails які треба надіслати на тижні 1. Це найдовший lead-time у проекті — відповіді можуть прийти через 1-4 тижні, тому **відправити негайно**.

## Як користуватись

1. Відкрий потрібний шаблон (`01-ittour.md`, `02-tbo-holidays.md`, `03-otpusk-tat.md`)
2. Заміни placeholder'и `{{YOUR_NAME}}`, `{{YOUR_EMAIL}}`, `{{YOUR_PHONE}}`, `{{ORACLE_RESERVED_IP}}` (якщо вже маєш) на свої дані
3. Скопіюй текст у свій email-клієнт (Gmail, Outlook, тощо)
4. Відправ зі **свого** email (від ім'я того хто буде керувати акаунтом)
5. У відповіді — `Reply-All` до мене (пересилка тексту відповіді), щоб я задокументував і просунувся далі

## Priority order

1. **#1 ittour** — критичний, основне джерело даних. Якщо відмовлять — нам потрібен план Б.
2. **#3 Otpusk/TAT.ua** — паралельний backup
3. **#2 TBO Holidays** — менш термінове (потрібно тільки для готельного контенту, можна почати без них)

## Що тримати в голові
- Це **B2B відносини**. Чим більш конкретно ти опишеш план — тим вище шанс відповіді.
- Не обіцяй того чого не можеш зробити (велика аудиторія, гарантовані обороти).
- Готов до того що відповідь буде "потрібно підписати договір і внести депозит $X" — це нормально, обговорюватимемо коли отримаємо.
- Якщо хтось не відповідає 14 днів — robotic follow-up з 1 reminder.
